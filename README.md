# 🚦 Traffic Sign Detection using Machine Learning

## 📌 Project Overview

This project is a **Traffic Sign Detection System** built using Machine Learning and Deep Learning techniques.
It can identify and classify different traffic signs from images, helping in real-world applications like **self-driving cars and driver assistance systems**.

---

## 🎯 Features

* 🔍 Detects traffic signs from uploaded images
* 🤖 Deep Learning model for accurate prediction
* 🖥️ User-friendly interface (GUI)
* ⚡ Fast and efficient processing

---

## 🛠️ Technologies Used

* Python 🐍
* TensorFlow / Keras
* OpenCV
* NumPy
* Streamlit (for GUI)

---

## 📂 Dataset

The dataset used in this project is taken from Kaggle:

(https://www.kaggle.com/datasets/meowmeowmeowmeowmeow/gtsrb-german-traffic-sign)

*(Download the dataset and place it inside a folder named `dataset/` before running the project)*

---

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Traffic-Sign-Detection.git
```

### 2️⃣ Navigate to project folder

```bash
cd Traffic-Sign-Detection
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the project

```bash
streamlit run streamlit_app.py
```
